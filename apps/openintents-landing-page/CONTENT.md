## Principles

### Accessibility

Making interoperability accessible and usable for chains, developers and users.

### Choice

### Trust-minimisation

## Solutions

The Open Intents initiative develops open standards, contracts, tooling and documentation.

### Standards

-   **ERC-7683** — An interface for cross-chain intent representation (v2 under development)
    → https://eips.ethereum.org/EIPS/eip-7683
-   **ERC-7930** — Standardising onchain binary representation of chain-specific addresses
    → https://eips.ethereum.org/EIPS/eip-7930
-   **ERC-7828** — Standardising a human-readable format for chain-specific addresses
    → https://eips.ethereum.org/EIPS/eip-7828

### Infrastructure

-   **Broadcaster** — Trustless storage-based message broadcasting for rollups
    → https://github.com/openintentsframework/broadcaster
-   **on.eth Chain Registry** — A canonical, ENS-native registry for chains and their metadata
    → https://ens.domains/blog/post/on-eth-chain-registry
-   **The Compact** — An ownerless ERC6909 contract that facilitates the voluntary formation and mediation of reusable resource locks
    → https://github.com/Uniswap/the-compact

### Framework (OIF)

-   **oif-contracts** — Modular locking, oracles and settlement
    → https://github.com/openintentsframework/oif-contracts
-   **oif-solver** — Cross-chain solver implementation for the Open Intents Framework
    → https://github.com/openintentsframework/oif-solver
-   **oif-specs** — Standardised interfaces for getting quotes and submitting orders
    → https://github.com/openintentsframework/oif-specs

### SDK (interop-sdk)

-   **addresses** — Easily interact with chain-specific addresses
    → https://docs.interop.wonderland.xyz/addresses
-   **cross-chain** — Access the OIF and other interoperability providers
    → https://docs.interop.wonderland.xyz/cross-chain

_We are constantly working to add more open standards and tools to improve the usability of interop on Ethereum._

## Open Intents for You

### For Chains

Deploy OIF contracts, The Compact, and Broadcaster, run OIF solvers to connect with the Ethereum ecosystem.

### For Infrastructure Providers

Integrate the OIF, add support for cross-chain addresses, leverage trust-minimised settlement solutions.

### For Interoperability Protocols

Integrate onchain and API standards, get supported in the interop-sdk.

### For Wallets & Applications

Integrate the interop-sdk to add cross-chain and interoperable addresses to your application.
