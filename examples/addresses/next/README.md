# Interopable Addresses - Next.js Example

Next.js template for building an interactive web application to demonstrate ERC-7930 interopable address conversion and validation.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Development

```bash
# Format code
pnpm format

# Lint code
pnpm lint:fix
```

## Next Steps

When implementing the example:

1. Add `@defi-wonderland/interop-addresses` to dependencies
2. Implement address conversion UI components in `app/`
3. See [Addresses Package](../../../packages/addresses/) and [Full Documentation](../../../apps/docs/docs/addresses/) for API usage
