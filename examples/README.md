# Examples

Working examples demonstrating Interop SDK functionality.

## Available Examples

### [addresses/](./addresses/)

Convert and validate ERC-7930 interopable addresses.

-   `next/` - Next.js web application
-   `cli/` - Command-line tool (coming soon)

## Quick Start

```bash
cd examples/addresses/next
pnpm install
pnpm dev
```

## Requirements

-   Node.js 20+
-   pnpm 9+
