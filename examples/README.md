# Examples

Interactive web application demonstrating Interop SDK functionality.

## [UI Demo](./ui/)

Next.js web application showcasing all SDK features:

-   ERC-7930 interopable addresses

## Quick Start

```bash
cd examples/ui
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Requirements

-   Node.js 20+
-   pnpm 9+
